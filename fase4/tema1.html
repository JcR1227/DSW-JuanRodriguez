Fundamentos de HTML

HTML es un lenguaje de programación que mediante etiquetas (< >) genera un sitio web. Los navegadores web reconocen e interpretan las etiquetas para mostrar todo el contenido (imágenes, texto) al usuario. Como un archivo HTML es solo texto, este debe redactarse en un editor de texto, por ejemplo: Visual Studio Code.
Si hablamos de las etiquetas, estas se organizan por pares, etiqueta de apertura y etiqueta de cierre. De igual forma, todas las páginas creadas con HTML conservan una estructura básica igual pues contienen las siguiente etiquetas: Doctype, HTML, Head, Title, Meta, Body.
Otras etiquetas que también se utilizan con gran frecuencia son: h1, br, p, hr, ul, ol.
(Celaya Luna, 2015)
